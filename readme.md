# Accessible Design Website (SBA Week 2)
## Description
The task was to design and develop a responsive, multi-section fictitious webpage meeting a few guidelines, including using semantic HTML, a responsive design  using flexbox and grid, and accessibility features using ARIA roles.
## Instructions to view
Using ARIA roles, flexbox and grid along with @media accessibilities allows this webpage to be viewed on variable devices with ease. Give your device, or preferred browser a go and see how it looks to you.
